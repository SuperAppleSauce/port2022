5G Home Self Setup

The Opportunity

Current 5G Home (C-Band) users struggle with troubleshooting & experience confusion of with signal testing. Some struggle to find next steps in the My Verizon App or faced technical issues in accessing the correct flow. Behaviors & observations suggest that the in app setup flow is helpful for some user archetypes. While other expect the device to work with minimal setup.

How might we...

Help users need help finding the right place to setup up their new 5G Home device without relying too much on the in app setup flows.

Testing signal finding without an app

Concept 1

Uses a familiar compass pattern found in app today. Allows users to orient using location data their device towards the nearest 5G cell.

Concept 2

Uses a pattern more common with WiFi technology. Allows users to place a device based on signal strength.

Findings

Users were able to comprehend the both concepts for signal finding. Users preferred the WiFi style concept more as it was most familiar. A few participants found that compass concept was too similar location services & caused concern.
